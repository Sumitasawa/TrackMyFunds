# Expense Tracker 

A **secure, real-time expense tracking web application** built using **React.js and Firebase**, designed to help users manage income, expenses, and spending patterns efficiently.

---

##  Features

- **Secure Authentication**
  - Google Sign-In using Firebase Authentication
  - Ensures safe and authorized user access

- **Real-Time Data Synchronization**
  - Firebase Firestore for real-time storage and updates
  - Data syncs instantly across sessions and devices

-  **Interactive Financial Dashboards**
  - Visualized income, expenses, and spending trends
  - Built using **@ant-design/charts**

- **Income & Expense Management**
  - Full CRUD operations (Create, Read, Update, Delete)
  - Real-time updates without page refresh

-  **Category-wise Expense Tracking**
  - Track spending by categories
  - Monthly summaries for better financial analysis

-  **Optimized Performance**
  - Efficient frontend state management
  - Smooth UI during frequent data updates

---

## Tech Stack

**Frontend**
- React.js
- JavaScript (ES6+)
- Ant Design Charts

**Backend / Services**
- Firebase Authentication
- Firebase Firestore
